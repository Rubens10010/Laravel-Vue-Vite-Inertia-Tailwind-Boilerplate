# MAIN FEATURES

-   Laravel 9+
-   Vue 3 + Typescript
-   Vite
-   InertiaJs
-   Tailwind

# INSTALLATION

-   git clone this.repo.git project
-   cd project
-   cp .env .env.example
-   composer install
-   php artisan key:generate
-   npm install
-   npm run dev

# DEVELOPMENT

-   npm run dev
-   php artisan serve
-   open VITE_APP_URL

# PRODUCTION

-   npm run build
-   deploy project to web server [apache/nginx]

# SERVE FROM WSL

-   php artisan serve --host=$ip_wsl
